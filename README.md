# LSTM Electricity Consumption Prediction

This repository contains an LSTM-based model for predicting
monthly electricity consumption (kWh).

## Input Features
- Electricity consumption of the previous month (kWh)
- Number of holidays in the target month

## Output
- Predicted electricity consumption (kWh)
- Estimated electricity bill (optional)

## Model
- LSTM (Lightweight)
- MinMax normalization
- Early stopping

## Notes
The dataset used for training is not included in this repository
for data privacy and academic purposes.
